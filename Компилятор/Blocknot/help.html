Справка по текстовому редактору

Меню:

  • Основные функции
    • Создать +

      Создать — Создает новый файл.

    • Открыть +

      Открыть — Открывает существующий файл.

    • Сохранить +

      Сохранить — Сохраняет текущий файл.

    • Сохранить как +

      Сохранить как — Сохраняет текущий файл под новым именем.

    • Отменить +

      Отменить — Удаляет последнее введенное слово.

    • Повторить +

      Повторить — Возвращает последнее удаленное слово.

    • Вырезать +

      Вырезать — Вырезает выделенный текст.

    • Копировать +

      Копировать — Копирует выделенный текст.

    • Вставить +

      Вставить — Вставляет текст из буфера обмена.

    • Удалить +

      Удалить — Удаляет выделенный текст.

    • Выделить все +

      Выделить все — Выделяет весь текст в редакторе.
